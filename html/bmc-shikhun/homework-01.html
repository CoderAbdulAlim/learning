01. What is Block Element?

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can). Two commonly used block elements are: p and div

02. What is Inline Element?

Inline elements display in a line. They do not force the text after them to a new line. An anchor (or link) is an example of an inline element. You can put several links in a row, and they will display in a line.

05. Common attribute list for HTML

Attribute Related Tag Description
alt area, img, input alternate text when the original element fails to display
colspan td, th number of columns a table cell should span
disabled button, fieldset, input, optgroup, option, select, textarea specified element/group of elements should be disabled
for label, output which form element(s) a label/calculation is bound to
form button, fieldset, input, label, meter, object, output, select, textarea name of the form the element belongs to
formaction button, input where to send the form-data when a form is submitted. Only for type="submit"
headers td, th one or more headers cells a cell is related to
height canvas, embed, iframe, img, input, object, video height of the element
href a, area, base, link URL of the page the link goes to
hreflang a, area, link language of the linked document
label track, option, optgroup title of the text track
max input, meter, progress maximum value
maxlength input, textarea maximum number of characters allowed in an element
min input, meter a minimum value
name button, fieldset, form, iframe, input, map, meta, object, output, param, select, textarea name of the element
onload body, iframe, img, input, link, script, style Script to be run when the element is finished loading
readonly input, textarea element is read-only
rel a, area, form, link relationship between the current document and the linked document
required input, select, textarea element must be filled out before submitting the form
rowspan td, th number of rows a table cell should span
size input, select width, in characters (for input) or number of visible options (for select)
sizes img, link, source size of the linked resource
span col, colgroup number of columns to span
src audio, embed, iframe, img, input, script, source, track, video URL of the media file
target a, area, base, form target for where to open the linked document or where to submit the form
type a, button, embed, input, link, menu, object, script, source, style type of element
value button, input, li, option, meter, progress, param value of the element
width canvas, embed, iframe, img, input, object, video width of the element

Attribute Related Tag Description

class Global Attributes one or more classnames for an element (refers to a class in a style sheet)
cols textarea visible width of a text area
content meta Gives the value associated with the http-equiv or name attribute
hidden Global Attributes an element is not yet, or is no longer, relevant
high meter range that is considered to be a high value
id Global Attributes a unique id for an element
list input Refers to a datalist element that contains pre-defined options for an input element
low meter range that is considered to be a low value
method form HTTP method to use when sending form-data
onblur All visible elements. Script to be run when the element loses focus
onchange All visible elements. Script to be run when the value of the element is changed
onclick All visible elements. Script to be run when the element is being clicked
onfocus All visible elements. Script to be run when the element gets focus
onkeypress All visible elements. Script to be run when a user presses a key
onreset form Script to be run when a reset button in a form is clicked.
onscroll All visible elements. Script to be run when an element's scrollbar is being scrolled
onsearch input Script to be run when the user writes something in a search field (for input type="search")
onselect All visible elements. Script to be run when the element gets selected
onsubmit form Script to be run when a form is submitted
rows textarea visible number of lines in a text area
selected option an option should be pre-selected when the page loads
style Global Attributes inline CSS style for an element
title Global Attributes extra information about an element
wrap textarea how the text in a text area is to be wrapped when submitted in a form
